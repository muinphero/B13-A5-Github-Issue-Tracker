## JavaScript Questions

### 1. Difference between var, let, and const

- `var` is function-scoped and can be redeclared.
- `let` is block-scoped and can be reassigned, but not redeclared in the same block.
- `const` is block-scoped and cannot be reassigned after declaration.

### 2. What is spread operator (...)?

The spread operator `...` expands values from arrays/objects.

- For arrays: copy or merge arrays.
- For objects: copy or merge object properties.

### 3. Difference between map(), filter(), and forEach()

- `map()` creates a new array by transforming each item.
- `filter()` creates a new array with only matching items.
- `forEach()` runs a function for each item but does not return a new array.

### 4. What is an arrow function?

An arrow function is a shorter way to write functions using `=>`.
It is commonly used for callbacks and small functions.

### 5. What are template literals?

Template literals use backticks (`` ` ` ``) to create strings.
They allow variable interpolation with `${}` and support multi-line strings.
